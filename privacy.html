EZkanji-chrome-ext Privacy Policy / プライバシーポリシー

1. Extension Purpose / 拡張機能の目的

This extension has a single, clear purpose: to automatically add Hiragana (ruby annotations) to Japanese kanji characters on webpages, making Japanese text easier to read and understand.
本拡張機能の唯一の目的は、Webページ上の日本語の漢字に自動的にふりがな（ひらがな）を付けることにより、日本語の読解を補助することです。

2. Data Collection / データ収集について

EZkanji-chrome-ext does not collect, store, or transmit any personal information or browsing data. All processing is done locally in the user's browser.
本拡張機能は、個人情報や閲覧データを一切収集・保存・送信しません。すべての処理はユーザーのブラウザ内でローカルに行われます。

3. Permissions / 使用する権限

- activeTab: Used only when the user clicks the extension icon to access the current page and apply ruby annotations.
ユーザーが拡張機能のアイコンをクリックしたときに、現在のタブにふりがな処理を行うために一時的に使用します。

- storage: Used to save the user's settings (such as on/off toggle) locally in the browser.
機能のオン・オフ設定などをユーザーのブラウザにローカル保存するために使用されます。

- host permissions: Required to read webpage content to process kanji and inject annotations.
ページの漢字にふりがなを付けるために、閲覧中のWebページの内容にアクセスする必要があります。

4. Remote Code & Third-party Services / リモートコードと外部サービス

This extension does not use any remote code or communicate with external servers.
本拡張機能はリモートコードや外部サーバーとの通信を一切行いません。
